# Supported Motion Lab prompt

## Input

Open the public Motion Control Lab and submit:

```text
roundhouse kick
```

## Expected boundary

- The lab must show `SYSTEM READY` before a command is submitted.
- The command resolves locally to the `roundhouse` animation label.
- The browser plays a local FBX clip only.
- No physical robot, repository code, wallet, email, transaction, or payment is
  involved.
